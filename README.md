# RashidKhanApDev.github.io 🌐

This is my personal developer portfolio and web showcase, built with pure HTML, CSS, and JavaScript.

## ✨ Features
- **Premium Glassmorphism UI:** Built with modern CSS techniques including `backdrop-filter`, smooth transitions, and transparent components.
- **Responsive Design:** Completely responsive layout that looks great on mobile, tablet, and desktop.
- **Enterprise Ready:** Configured with GitHub Actions (CodeQL Security), Dev Containers, Copilot strict instructions, and Stale bot automation.

## 🛠️ Tech Stack
- **HTML5:** Semantic architecture.
- **CSS3:** Advanced Glassmorphism and CSS variables.
- **JavaScript:** Encrypted URL payload parsing.

## 🚀 Development (Codespaces)
This repository is optimized for **GitHub Codespaces**. You can start coding immediately in a cloud environment:
1. Click the green **Code** button.
2. Select the **Codespaces** tab.
3. Click **Create codespace on main**.
